## 📦 Getting Started

- Clone the repository:
```sh
git gh repo clone yasirmansoori/100DaysOfCode
```
- Navigate to this project directory:
```sh
 cd Day 9
```
- Install dependencies: 
```sh
 npm install
```
- Start the development server:
```sh
  npm run dev
```
